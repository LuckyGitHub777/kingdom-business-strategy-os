# Roadmap

This roadmap separates what is present in the current release from what is intended for future versions.

## Present in v0.2.3

- Core business doctrine
- Executive mastery framework
- Business-simulation-to-real-business translation
- Optional advanced frontier-business track
- Frontier readiness starter checklists
- Mission and stewardship alignment prompts
- Quickstart
- Operating cadence
- Canonical taxonomy
- Six templates
- Six worked examples
- Security/contact policy
- Contribution workflow

## Planned for v0.3.0

- SOP library
- Dashboard starter pack
- Risk register template
- AI agent governance checklist
- Vendor questionnaire template
- Board/executive briefing template
- Public-safe evidence package template
- More industry-specific examples
- Expanded public-safe frontier examples

## Planned for later versions

- Lightweight metrics tracker
- Quarterly business review packet
- AI automation readiness audit template
- Quantum-safe readiness score template
- Cyber-AI governance pack
- Evidence governance framework for high-uncertainty claims

## Boundary

Items listed as planned are not yet shipped. Do not treat roadmap items as current deliverables.
